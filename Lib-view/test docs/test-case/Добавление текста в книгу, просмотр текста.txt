Тест-кейс 1
Название: Добавление текста в книгу
Предусловие : войти в ЛК ( валидный логин и имейл)

Шаги:
1. Открыта страница сайта: http://127.0.0.1:8080/private/add-b
2. Выбрать книгу в которую нужно добавить текст
3. Выбрать пункт "Добавить текст"
4. В поле ввода "Добавить текст" ввести текст с разметкой, символы разметки 1- {  ,  2- | , 3-  {    
5. Нажать на кнопку "Сохранить"
6. Сообщение о результате

Ожидаемый результат: Текст добавлен в книгу

Тест-кейс 2
Название: Просмотр текста книги
Предусловие : войти в ЛК ( валидный логин и имейл)

Шаги:
1. Открыта страница сайта: http://127.0.0.1:8080/private/add-b
2. Выбрать книгу в которую нужно просмотреть текст
2. Выбрать пункт "Просмотреть текст"

Ожидаемый результат: Текст отображается корректно
