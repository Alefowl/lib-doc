ADA01
Название:
Проверка добавления автора

Предусловие:
Пользователь авторизован на сайте Alefowl.com и находится в личном кабинете

Шаги:
1. Нажать на кнопку "Add Author"
2. Заполнить поля имя и фамилия автора на всех языках (английский, русский, немецкий, французский) "Login"
3. Заполнить поля Дата рождения и дата смерти (только год)
4. Из выпадающего списка выбрать родной язык автора
5. Нажать кнопку "Add Author"

Ожидаемый результат:
Появляется окно с информацией об успешном добавлении автора
