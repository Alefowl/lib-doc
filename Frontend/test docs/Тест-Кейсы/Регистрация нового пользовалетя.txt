ID 
TK01

Название
Проверка регистрации пользователя (ввод валидных данных)

Шаги
1. Зайти на сайт http://localhost:3000/
2. Нажать кнопку "Login" в правом верхнем углу  
3. Нажать кнопку Don't have an account? Sign Up
3. Заполнить поля "Login", "First Name", "Surname", "Email", "Password" - валидными данными                                  
4. Нажать кнопку "Register"

Expected Result/ Ожидаемый результат
1. Пользователь зарегестрирован на сайте и на почту пришло письмо с подтверждением регистрации
