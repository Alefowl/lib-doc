ID 001
Название: Регистрация с валидными данными
Шаги:
1. Перейти на страницу http://localhost:3000/registration
2. Ввести валидные данные в поля регистрации
3. В поле login ввести «goolsy»
4. В поле First Name ввести «Оксана»
5. В поле Surname ввести «Рудая»
6. В поле Mail ввести goolsy@mail.ru
7. В поле Password ввести 961Ah1234
8. Нажать «Register»
ОР: аккаунт успешно создан, появляется попап «successfully created»
