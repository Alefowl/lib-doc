Название: Добавление записи в таблицу Chapters (корректные данные с токеном авторизации)

Номер: UCH-001

Шаги:
1.Открыть Postman.
2.Выбрать метод запроса: POST.
3.В поле URL ввести: http://localhost:8080/admin/insertChapter.
4.В разделе "Body" выбрать опцию "raw" и установить формат "JSON (application/json)".
5.Вставить следующий JSON-запрос в поле Body:
{
  "id": "6",
  "book": "2",
  "title": "sdcsdc",
  "txt": "ыавиваывам"
}
6.В хедерах запроса добавить следующий заголовок:
Authorization: token
7.Нажать на кнопку "Send" для отправки запроса.

Ожидаемый результат:
Статус код ответа должен быть 200 (OK).
Тело ответа должно содержать значение "1", что указывает на успешное добавление записи в БД.



Название: Добавление записи в таблицу Chapters (корректные данные, без токена авторизации)

Номер: UCH-002

Шаги:
1.Открыть Postman.
2.Выбрать метод запроса: POST.
3.В поле URL ввести: http://localhost:8080/admin/insertChapter.
4.В разделе "Body" выбрать опцию "raw" и установить формат "JSON (application/json)".
5.Вставить следующий JSON-запрос в поле Body:
{
  "id": "6",
  "book": "2",
  "title": "sdcsdc",
  "txt": "ыавиваывам"
}
6.В хедерах запроса не передавать токен авторизации.
7.Нажать на кнопку "Send" для отправки запроса.

Ожидаемый результат:
Статус код ответа 400 (Bad Request).
Тело ответа должно содержать сообщение об ошибке авторизации.
