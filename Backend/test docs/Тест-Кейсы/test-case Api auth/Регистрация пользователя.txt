Название: Регистрация пользователя (корректные данные)

Номер: RG-001

Шаги:
1.Открыть Postman.
2.Выбрать метод запроса: POST.
3.Перейти на страницу http://localhost:8080/registration.
4.В разделе "Body" выбрать опцию "raw" и установить формат "JSON (application/json)".
5.Вставить следующий JSON-запрос в поле Body, содержащий неверные данные:
  {
    "login": "login0000001",
    "password": "password"
    "e-mail":"fhfhshjsifhfj@gmail.com"
  }
6.Нажать на кнопку "Send" для отправки запроса.

Ожидаемый результат:
Статус код ответа 200 OK.
