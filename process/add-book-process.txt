Алгоритм добавления книги.
0. Предварительными условиями добавления книги является зарегистрированный пользователь со статусом Administrator, 
от лица которого происходит добавление книги на стенд.
1. Если автора нет в системе, то необходимо создать нового автора на странице Add author в личном кабинете пользователя.
2. Если метакниги нет в системе, то необходимо создать нового автора на странице Add metabook в личном кабинете пользователя.
   В метакниге должен быть автора из предыдущего пункта
3. Создать новую книгу на основе метакниги из предыдущего пункта на странице Add book в личном кабинете пользователя.
4. Добавить текст книги в созданную в пункте 3 книгу в соответствующее поле на странице Add text в личном кабинете пользователя.
5. После успешного добавления книги  на стенд надо проверить, что минимум 2 перевода текста читаются в параллель. 
Также желательно проверить корректно ли скачивается книга в PDF.

Правила разметки книги.
a.	Текст размечается с помощью символов {, |, [], @.
b.	Разметка главы: { - это начало главы, | - это конец заголовка главы 
c.	Разметка сносок: [Номер сноски] [Номер сноски] текст сноски @. Сноски размечаются прямо в тексте.
d.	Между абзацами в тексте главы должно быть 2 переноса строки. То есть между двумя абзацами должна быть пустая строка.  
e.      Размеченный текст завершается символом {
Важно:      Число глав в текстах книг, ассоциированных с одной метакнигой, должно совпадать.
  Пример размеченного текста:
{ Часть 1 | 
{ Глава 1 | 
Текст главы 1
{ Глава 2 | 
Текст главы 2
{ Часть 2 | 
{ Глава 1 | 
Текст главы 1
{ Глава 2 | 
Текст главы 2
{ Глава 3 | 
Текст главы 3
 {
Пример размеченной сноски:
[1][1] текст сноски @

Примечание 1: Если по какой-то причине совершена ошибка с разметкой, добавлением книг/метакниг/авторов, добавлением текста,  
книгу надо удалить и добавить заново, исправив ошибки. 
Удаление книги осуществляется на странице “My books” в личном кабинете пользователя с помощью кнопки “Delete book”.

Примечание 2: При добавлении переводов книги, желательно указывать верный год перевода и сохранять у себя переводчика книги.
